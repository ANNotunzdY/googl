= googl

Google URL Shortener API in Ruby

== Basic Usage

=== Shorten a long URL

  url = Googl.shorten('http://www.zigotto.com')

  url.short_url
  => "http://goo.gl/ump4S"

  url.long_url
  => "http://www.zigotto.com/"

=== Expand a short URL

  url = Googl.expand('http://goo.gl/ump4S')

  url.long_url
  => "http://www.zigotto.com/"

== Installation

  gem install googl
